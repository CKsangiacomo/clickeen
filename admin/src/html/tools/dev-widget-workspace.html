Widget Workspace

Zero-to-one workspace

Open Bob from current source defaults or an existing admin starter. This is the narrow DevStudio loop for widget birth work, not a local Roma clone.

Source defaults mode is in-memory authoring from tokyo/widgets/{widget}/spec.json. Starter mode opens a real curated or baseline row under the admin account so Bob keeps its normal save/publish/localization behavior.

Open starter in new tab

Widget Workspace is local-first. Run bash scripts/dev-up.sh and open http://localhost:5173 for the default loop.

If you want hybrid dev from a remote DevStudio surface, pass ?bob=http://localhost:3000 on the page URL so the workspace can attach to your local Bob/Paris/Tokyo stack.
