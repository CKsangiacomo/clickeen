STATUS: PRD 101 STUB - STATIC BUILD ONLY

PRD 100 removes runtime embed composition. SEO/GEO is no longer a Venice loader behavior or a public render-projection read.

```txt
save -> Translation Agent Worker (its own home) -> thin typed locale overlays via Tokyo-worker -> clk.live
```

SEO/GEO optimization is a paid build shape that PRD 101 will define.
```txt
https://clk.live/{accountPublicId}/{instanceId}
```
