# PRD 067 — Internal Control Plane

Status: PENDING
Date: 2026-03-13
Owner: Product Dev Team
Priority: P0

## One-line objective

Define and build the separate internal control plane for company-plane operations such as support intervention, trust and safety actions, commercial overrides, and internal account moderation.

## Scope note

This PRD is the direct follow-up to PRD 066.
It exists because DevStudio remains the internal toolbench while Berlin remains the canonical product identity/account boundary.
The future company-plane authority model must therefore be defined separately instead of being collapsed into Berlin product roles.
